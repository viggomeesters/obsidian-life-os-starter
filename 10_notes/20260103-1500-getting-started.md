# Getting Started with Life OS

## Daily Workflow

1. Open daily note (Calendar sidebar or command palette)
2. Review schedule and tasks for the day
3. Log interactions, health events, and purchases as they happen
4. End of day: quick review of what was accomplished

## Weekly Workflow

1. Create weekly note (optional)
2. Review completed tasks
3. Plan next week's priorities
4. Check on project progress
