# Welcome to Life OS

Welcome to your new vault! This starter template gives you a fully configured Life OS v7.4 setup.

## First Steps

1. **Install community plugins**: Open Settings → Community Plugins → Browse, and install these 10 plugins:
   - Templater
   - Periodic Notes
   - Obsidian Git
   - Calendar
   - Minimal Theme Settings
   - Style Settings
   - Hider
   - Recent Files
   - Iconize
   - Settings Search

2. **Install the Minimal theme**: Settings → Appearance → Themes → Browse → search "Minimal" → Install and Use

3. **Enable plugins**: After installing, enable all 10 plugins in Settings → Community Plugins

4. **Restart Obsidian**: Close and reopen the vault to apply all configurations

5. **Create your first daily note**: Use the Calendar plugin in the right sidebar or run "Periodic Notes: Open daily note" from the command palette (Alt+P)

## Key Concepts

- **12 note types**: entity, interaction, purchase, anniversary, health, entry, task, project, reference, chore, ritual, context
- **Flat structure**: All timestamped notes in `10_notes/`, context docs in `20_structure/`
- **WikiLinks**: Always use `[[slug]]` format, never paths or extensions
- **Frontmatter**: Every note has structured YAML metadata
